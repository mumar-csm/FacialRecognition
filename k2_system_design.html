Endpoints invoke shared recognition pipeline components
⚙ Recognition Pipeline K2
1Decode
base64 → bytes → cv2.imdecode() → BGR array
↓
2Detect Faces
RetinaFaceDetector.detect(frame) → [(bbox, landmarks)]
0 faces → "no_face"

>1 faces → "multiple_faces"
↓ exactly 1 face
3Anti-Spoof
MiniFASChecker.check(frame, bbox) → (is_real, score)
is_real=False → "spoof_detected"
↓ real face
4Align + Embed
align_face() → 112×112 → ArcFaceEmbedder → 512-D vec
↓
5Match
euclidean_distance(emb, known_embs) → name, dist
dist > threshold → "unknown"
↓ match found
6Cooldown Check
cooldown_dict[name] — suppress if < N min
In cooldown → "recognized" + suppressed:true
↓ not in cooldown
7Log & Respond
INSERT → attendance.db  |  update cooldown_dict
"recognized" + identity + distance
⚙ Enrollment Pipeline K3
1Decode
base64 → bytes → cv2.imdecode() → BGR array
↓
2Detect Faces
RetinaFaceDetector.detect(frame) → [(bbox, landmarks)]
0 faces → "no face found"

>1 faces → "multiple faces — use single portrait"
↓ exactly 1 face
3Anti-Spoof
MiniFASChecker.check(frame, bbox) → (is_real, score)
is_real=False → "use a real face, not a photo"
↓ real face
4Align + Embed
align_face() → 112×112 → ArcFaceEmbedder → 512-D vec
↓
5Save Photo
cv2.imwrite() → data/employees/{employee_id}.jpg
↓
6Update Database
Append encoding to known_faces_arcface.pkl
↓
7Hot-Reload
Reload EncodingsDB in memory — no restart needed
{ success: true } + face_crop thumbnail
